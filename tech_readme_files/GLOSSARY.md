# Glossary

| Term | Meaning |
|------|---------|
| **messageKey** | Stable i18n key in API responses (e.g. `auth.login.success`) |
| **Idempotency-Key** | Client header for safe offline retries |
| **Flutter-compatible JSON** | Trip/delivery payloads matching the mobile app's expected field names and status strings |
| **Eloquent** | ORM for PostgreSQL |
| **Laravel Queues** | Redis-backed job queue for notification retries |
